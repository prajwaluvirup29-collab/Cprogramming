# Remove Duplicate Elements from an Array

## Description
This program removes duplicate elements from an array and displays the unique elements.

## Algorithm
1. Read array elements.
2. Compare each element with the next elements.
3. Shift elements left when duplicates are found.
4. Reduce array size and print unique elements.

## Input
- Number of elements
- Array elements

## Output
- Array without duplicate elements

## Language
- C
