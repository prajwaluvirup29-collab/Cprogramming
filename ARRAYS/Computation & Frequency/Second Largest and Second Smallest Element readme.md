# Second Largest and Second Smallest Element in an Array

## Description
This program finds the second largest and second smallest elements
from a given array without sorting it.

## Approach
The program maintains four variables:
- largest
- secondLargest
- smallest
- secondSmallest

Each array element is compared and values are updated accordingly.

## Input
- Number of elements
- Elements of the array

## Output
- Second largest element
- Second smallest element

## Example
Input:
6  
10 5 8 20 2 15  

Output:
Second Largest Element: 15  
Second Smallest Element: 5
