# Count Frequency of Each Element in an Array

## Description
This program counts how many times each element appears in an array entered by the user.

## Algorithm
1. Read the array elements.
2. Compare each element with the remaining elements.
3. Count repetitions and mark counted elements.
4. Display each element with its frequency.

## Input
- Number of elements
- Array elements

## Output
- Each unique element with its frequency

## Language
- C
