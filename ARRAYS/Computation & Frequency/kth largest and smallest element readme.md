# Kth Largest and Kth Smallest Element

## Description
This program finds the Kth smallest and Kth largest elements in an array by sorting the elements in ascending order.

## Algorithm
1. Read array elements.
2. Sort the array.
3. Kth smallest = element at index (k-1).
4. Kth largest = element at index (n-k).

## Input
- Number of elements
- Array values
- Value of k

## Output
- Kth smallest element
- Kth largest element
