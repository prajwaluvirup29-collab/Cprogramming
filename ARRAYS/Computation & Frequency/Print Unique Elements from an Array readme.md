# Print Unique Elements from an Array

## Description
This program reads elements into an array and prints only the elements
that appear exactly once in the array.

## Logic Used
Each element is compared with all other elements.
If it does not match with any other element, it is considered unique.

## Input
- Number of elements
- Elements of the array

## Output
- Unique elements present in the array

## Example
Input:
5  
1 2 2 3 4  

Output:
1 3 4
