# Merge Two Arrays

## Overview
This program merges two different arrays into a third array without sorting.

## Steps
1. Read elements of the first array.
2. Read elements of the second array.
3. Copy both arrays into a third array.
4. Display the merged array.

## Input
- Size and elements of first array
- Size and elements of second array

## Output
- Combined array containing all elements

## Programming Language
C
