# Linear Search in C

## Description
This program performs a linear search on an array. It checks each element one by one until the required element is found or the array ends.

## Algorithm
1. Read the number of elements.
2. Read array elements.
3. Read the element to be searched.
4. Compare the search element with each array element.
5. If found, display its position.
6. Otherwise, display "Element not found".

## Advantages
- Easy to understand
- Works on both sorted and unsorted arrays

## Time Complexity
- Best Case: O(1)
- Worst Case: O(n)

## Language Used
- C Programming
