# Shift All Zeros to the End

## Description
This program moves all zero elements of an array to the end while maintaining the order of non-zero elements.

## Algorithm
1. Read array size and elements.
2. Copy all non-zero elements into a new array.
3. Fill remaining positions with zeros.
4. Print the updated array.

## Input
Number of elements and array values.

## Output
Array with all zeros shifted to the end.

## Language
C
