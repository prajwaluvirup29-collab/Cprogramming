# Rotate Array by K Positions (C Program)

## Overview
This program rotates an array by a user-defined number of positions (k)
using an auxiliary array for simplicity.

## Working Principle
- The modulo operator ensures valid index positions.
- A temporary array stores rotated values.

## Input Format
- Size of array
- Array elements
- Value of k (rotation count)

## Output Format
- Rotated array

## Advantages
- Simple logic
- Avoids complex swapping
- Easy to understand for beginners

## Programming Language
C
