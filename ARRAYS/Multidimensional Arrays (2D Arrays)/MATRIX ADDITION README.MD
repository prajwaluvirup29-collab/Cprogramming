# Matrix Addition Using User-Defined Size in C

## Description
This program performs matrix addition by allowing the user
to enter the number of rows and columns. The program then
adds two matrices of the same size and displays the result.

## Concepts Used
- 2D arrays
- User input
- Nested loops
- Matrix addition

## Input
- Number of rows and columns
- Elements of two matrices

## Output
- Displays the sum of the two matrices
